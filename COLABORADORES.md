# Equipo de Trabajo Clio Astra

¡Bienvenidos! Este es un ejercicio para practicar el flujo básico de Git y GitHub (clonar, hacer un commit, y subir cambios).

## Instrucciones para los colaboradores:
1. Clona este repositorio en tu computadora.
2. Abre este archivo (`COLABORADORES.md`).
3. Añade tu nombre, tu rol y algo que te guste en la lista de abajo (asegúrate de hacerlo en una nueva línea).
4. Guarda el archivo.
5. Haz un commit con tus cambios (`git add COLABORADORES.md` y `git commit -m "Agrego mi nombre a la lista"`).
6. Sube tus cambios a GitHub (`git push`).
